# PHP et MySQL avec Docker

## Prérequis

- [Docker](https://docs.docker.com/install/)
- [Docker Compose](https://docs.docker.com/compose/install/)
- [Composer](https://getcomposer.org/download/)

## Lancer le projet
```bash
docker-compose up
```

## Accéder au projet
```bash
http://127.0.0.1:8080/
```

## Accéder à phpMyAdmin
```bash
http://127.0.0.1:8081/
```
